# 🌊 Water Usage Tracker

**Water Usage Tracker** is a simple Python tool to help you log daily water consumption and visualize it.  
You can see your usage as **ASCII bars** in the terminal or as a **matplotlib chart**.

---

## Features

- 💧 Log daily water usage (in liters)  
- 📊 Visualize daily usage with ASCII bars  
- 📈 Optional matplotlib chart for better visualization  
- 💾 Stores data in `water_usage.json` in the current folder  
- 🖥️ Works in terminal or Python IDE

---
